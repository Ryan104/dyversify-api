# Dyversify

## Installation

- Install/start MongoDB `docker run --name mongo -d mongo:4 -p 27017:27017`
- Install `npm install`
- Rename `.env.example` to `.env`
- Run `npm run dev`
- Check `http://localhost:3000/api/status` to see it works
